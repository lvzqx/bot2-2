# Discord Bot 使い方

## 📝 投稿を作成
**/post**
ここに思ったことを書き込めます。
フォームが開くので、内容を入力して送信してください。

## 🔍 投稿を検索
**/search**
過去に書いた内容をキーワードで探せます。
カテゴリでの絞り込みもできます。
検索タイプ：投稿、リプライ、いいね
Google風の検索インターフェースで詳細表示も可能です。

## 📋 自分の投稿一覧
**/list**
自分が書いたものの一覧が見られます。
最新のものから順に表示されます。
Embed形式で詳細表示、ページネーション機能も搭載しています。

## ✏️ 投稿を編集
**/edit**
過去の投稿を修正できます。
一覧から選択すると編集できます。

## 🗑️ 投稿を削除
**/delete**
投稿を削除できます。
投稿IDを指定するか、一覧から選択して削除してください。

## 💬 リプライする
**/reply**
投稿に返信できます。
投稿IDとリプライ内容を入力してください。
元の投稿は「リプライ」チャンネルに転送され、Embed形式で表示されます。

## ✏️ リプライを編集
**/edit_reply**
自分のリプライを修正できます。
リプライIDを選択して編集してください。

## 🗑️ リプライを削除
**/delete_reply**
自分のリプライを削除できます。
元の投稿IDを指定して、自分のリプライと転送メッセージを削除してください。

## ❤️ いいねする
**/like**
投稿にいいねできます。
投稿IDを指定していいねしてください。
元の投稿にリプライとしていいねが送信されます。

## ❌ いいねを削除
**/unlike**
いいねを取り消せます。
投稿IDを指定して、投稿へのいいねメッセージを削除してください。

## ❔ ヘルプ表示
**/help**
使えるコマンドの一覧が見られます。
このメッセージを表示します。

---

💡 **投稿IDは /list で確認できます**
